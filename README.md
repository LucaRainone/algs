# ALGS - Another Less Grid System


## Features
* mobile first
* responsive
* auto clear rows
* easy customization
* lightweight (2.4KB gzipped)
* ie9+ compatible (ie6-8 with [Respond.js](https://github.com/scottjehl/Respond))


## Setup
load css and enjoy


## Usage
see [demo](https://albebonv.github.io/algs/)
